# ABDL Bladder Sim

ABDL Bladder Sim is a timed omorashi/ABDL routine simulator.
You pick a profile, run event cycles, and update pressure/saturation so the pacing matches your real urgency.

## Quick Start

1. Open `index.html` in your browser.
2. Click `Start Session`.
3. Pick a profile and confirm setup.
4. When alarms appear, click `Reveal` and follow guide steps.
5. Keep `Pressure` and `Saturation` updated as your state changes.

## What The Main Panels Do

- Session Control: start/stop your run and pause alarms for 30 minutes when needed.
- Bio-Logger: your truth input (pressure and saturation).
- Protection: what you are wearing, plus optional stash tracking.
- Maintenance: change, force release, mercy toggle.
- Log: live feed of events and outcomes.

## Bio-Logger How To Use

Treat the sliders as honest state tracking.

- Pressure: urgency intensity (how close you feel to leaking).
- Saturation: wetness level of current protection.

### Pressure 1-10 Guide

- 1-2 (0-20%): comfortable, no urgency.
- 3-4 (25-40%): aware, easy to ignore.
- 5 (50%): clear urge, events begin accelerating.
- 6-7 (60-75%): active holding effort, small leaks possible.
- 8 (80%): strong urgency, dribbles likely.
- 9 (90%): emergency zone, high accident risk.
- 10 (95-100%): on the edge / actively losing control.

### Saturation Quick Guide

- 0-30%: lightly used.
- 40-70%: noticeably wet.
- 80%+: heavy and squishy.
- 100%: change now.

## Profile Suggestions

- New player: `Rookie`.
- Full caregiver loop: `Babysitter`.
- Passive background flow: `NPT`.
- High intensity regression: `Dependent`.
- Manual pacing: `Chaos`.
- Dedicated hold challenge: `Omorashi`.

## Custom Profiles

Use `Custom Profiles (Create / Import / Export)` in Session Control.

- Import from file: `Import JSON`.
- Import from clipboard: `Import Clipboard`.
- Save and reuse in the same browser via local storage.

## Notes

- If prompts are too frequent, lower pressure and check hydration settings.
- If prompts are too sparse, raise pressure to match your current urgency.
- Use `Pause Alarm (30m)` during meetings or interruptions.
